Danilo Parisi Snake Documentation

Introduction

The Snake game is a classic game where the player controls a snake that must eat food to grow. The objective is to avoid hitting the walls or the snake's own body. The game was developed using HTML, CSS, JavaScript, and PHP. The project includes login management, registration, a home page, and the actual gameplay.

How the Game Works

Login and Registration

Access to the game is handled via a simple login and registration interface.
- Login: Users can log in with their username and password. If credentials are correct, they will be redirected to the home page.
- Registration (Sign Up): New users can register by providing a username, email, and password. Once registered, they can access the home page and play the game.

Home Page

Once logged in, users are directed to the home page where they can:
- Launch the Snake game.
- View this documentation and rules.
- Log out of their session.

The Snake Game

Once the game starts, the user controls the snake using the keyboard arrow keys.
- The snake moves continuously and grows longer every time it eats an apple.
- The game ends when the snake collides with a wall or with its own body.
- The current score and personal best record are displayed and saved for each user.

Important Note

If you have not logged in or registered, you will not be able to access the game screen and will be automatically redirected to the home page.

How to Play

After logging in, open the game page and use the arrow keys on your keyboard to control the snake's movement:

  • Up Arrow: moves the snake upward.
  • Down Arrow: moves the snake downward.
  • Left Arrow: moves the snake to the left.
  • Right Arrow: moves the snake to the right.
